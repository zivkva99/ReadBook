# Tanakh daily reading schedule

Chapter-a-day schedule for the Jewish Bible (Tanakh), Sunday–Thursday only, starting from
יהושע י״ט (Joshua 19) on 14.6.2026 through the end of דברי הימים ב׳ (2 Chronicles 36).

- `tanakh_schedule.csv` — raw data: book (Hebrew), chapter number, chapter in Hebrew numerals, date (d.M.yyyy).
- `tanakh_reading_schedule.html` — standalone searchable/filterable view of the same data, open directly in a browser.

724 chapters total, one per weekday (Sun–Thu), ending 21.3.2029.
